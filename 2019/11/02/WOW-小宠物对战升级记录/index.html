一. 插件准备

插件可从 Twitch 上下载。

rematch

核心插件之一，功能很多，还在探索当中。

可实现自动换队，升级序列等操作。

需注意：如果需要自动换队功能，需要对插件进行修改。

tdBattlePetScript

核心插件之一，用以实现宠物对战自动化，此处自动化指对于宠物对战制定合理的策略，编写 script 来实现无脑按同一按键进行宠物对战。

为帮助想要自制脚本的同学放上项目 GitHub 链接，插件的介绍以及 API 都可在其中查询。

二. 准备工作

小宠物对战升级所需的大部分宠物都可在拍卖行购得，但需注意自己至少有一个 25 级宠物才能直接购买添加 25 级宠物。

如果没有 25 级宠物，最快速的拿到 25 级宠物方案是升级三级要塞，在宠物小屋接任务，完成后可拿到一个直接提升宠物为 25 级的石头。

戴米多斯的仆从 （必备）

影月谷高地稀有掉落，稀有怪刷新时间在 1 ～ 3 小时，一天只有一次拾取，且掉落不定，建议直接拍卖行购买 25 级蓝色品质。约 20000G。

黄猫（必备）

艾尔文森林购买，可建联盟试玩号，出暴风城购买，但买到的为 1 级绿色品质，需升级升品。黄猫为整个方案比较关键的一只宠物，可以用 //TODO 替代。建议有条件直接在拍卖行购买 25 级蓝色品质。约 15000G。

腐化的小助手（可选）

一级拍卖行价格在 7000G 左右，可以买一级宠练到 25 级，与戴米，黄猫组队单局平均约为 50s。

夺日者微型哨兵（可选）

拍卖行价格在 5000G 左右，如果觉得腐化小助手太贵可以用此替代，与戴米和黄猫组队单局平均约为 60s。

邪恶之魂（可选）

Icc 掉落，拍卖行可以购买。人宠双练时带一级小宠用，与机械野猪二选一。

机械野猪（可选）

拍卖行购买，价格 2000 左右。人宠双练使用，与邪恶之魂二选一。

三. 1 ~ 110 升级方案

理论上霜火岭可以一直升级到 120 级，但是 110 级以后效率明显降低，110 ～ 120 大约需要 12 小时，而重量级只需大约一半的时间。推荐使用重量级野兽方案。

队伍配置分纯人物升级方案和二带一人宠双升方案，因为 110 级以后刷重量级野兽需要一些新宠，所以建议在升级的过程当中也升级一些以后要用的宠物。

快速到达霜火岭

建立一个 110 级试玩的术士号，两人利用血槌熔渣矿井副本的集合石拉试玩术士到达霜火岭，一起到达升级地点，术士起门，将待升级小号拉过来。

或者找法师帮忙开一个阿什兰的传送门，再找一个可以德拉诺飞行的基友带过去。

四. 110 ～ 120 升级方案

可选择的方案有许多，这里我只实验了一种，宠物相对比较平民，且速度不错，所以只列出这一种方案。

重量级野兽

重量级野兽是军团版本的一个世界任务，位置在风暴峡湾。任务需要挑战两位训练师，只要不去打牛，我们便可无限打熊。

准备工作

升级号需至少 110 级。完成军团版本的前置任务，前置任务比较复杂故只给出流程链接（只要有一个号做过就可跳过前置，直接到达拉然）。还需注意前置任务这里，因为奥格瑞玛现在位面混乱，非常容易出现无法找到任务 NPC 这种情况。只能通过联系 GM 填写表单解决。单个表单平均解决时间 20 小时。建议此处联系 GM 时直接请他帮忙推进到达拉然。

到达破碎群岛达拉然拿到炉石后去找卡德加，完成联合力量任务，解锁世界任务。

此时地图的鸟点是不解锁的，如果想要自己去风暴峡湾需要开启职业大厅，然后做完一个场景战役才能到达风暴峡湾，推荐使用位于风暴峡湾的英灵殿副本集合石，或者找一个开了飞行的大号带过去。

队伍设置

嚎叫离子炮队：

1号嚎叫宠 (281+速度的霜狼幼崽/狐狸等) + 2号嚎叫宠 (281+速度，与1号不同种类，1号死后替补) + 德莱尼微型防御者 (2轮，极速22秒，使用宠物最少的高速升级方案！只要1队)

只要1个队，并设为种子队(选中NPC-右键重命名或改变目标-目标选择“瑞迪尔”)。4-5只快速嚎叫宠即可 (2种不同嚎叫宠，各2只，或3+1，或3+2均可。1号位放一种，2号位放另一种。rematch选项里面 勾选“加载血量最高的宠物”，如果抓的宠物属性不同(比如SB和BB等混合)，下面“Allow Any Version”也打勾)，这样打完会自动换血量较高的同种宠物，不用换队，简单高效之极。

)

狐狸可以去风暴峡湾、至高岭，昆莱山等地图抓，281+速度的，抓3只名字一样的(嚎叫宠，总共3-5只就够用)。德莱尼理论上可以只用一个，由于是机械，永远不死；只是3个更效率，大大减少机械复生动画时间。

德莱尼微型防御者价格在 10000G 左右，这里理论上可以用其他有离子炮技能的宠物代替。但我没有多余宠物，此处未经测试。

技能随意，只需保证有嚎叫和离子炮即可。

如果用其他宠物请将代码中的德莱尼微型防御者替换为宠物名字，其余可保持不变。
