Mac软件破解研究

本篇文章仅作为研究目的，文章中涉及的软件我已都不再使用，仅仅是一时兴起想要看看各个厂商都是如何保护自己的试用版。

现在的好多软件商都会提供试用版的软件，这就留下了一些空子可以钻。主要方法是修改软件的 plist 文件，plist 文件存储了一些软件的设置，具体就不展开了。只要能找到 plit 中的一些关键信息，如激活日期，到期日期等，就可以通过修改这些信息来达到破解的目的。

使用的工具为 PlistEditor Pro，这款软件可以实现修改 plist 文件的功能，plist 文件大部分存于 Library 中的 Preference 中。PlistEditor 本身就提供使用版，但是过期之后并不影响使用。

目前测试使用到如下软件。

Things 3

方法一：如果你从来没有安装过这个trail版本，直接把当前系统时间改成未来的某个时间，比如2020年，然后安装，运行一次。之后退出程序，将系统时间还原。恭喜你，你已经破解完成了。
方法二：如果你不幸已经安装并运行过软件，那就需要找到安装记录并删除它。步骤如下：
1，显示系统隐藏文件。
在终端（Terminal）中输入如下命令：defaults write ~/Library/Preferences/com.apple.finder AppleShowAllFiles -bool true
（true 改成 false 就可以不再显示隐藏文件）需要重启，然后在finder中就可以看到隐藏文件了。

command + shift + . 可以直接显示隐藏文件。

2，打开~/Library/Preferences/ByHost下的GlobalPreferences.XXXX.plist文件，XXXX为一串数字，删掉”com.zennaware.Cornerstone:2.0”这一行，然后保存。（如果安装了XCode，会自动打开plist文件，否则就得自己找个plist文件工具了，比如PlistEdit）
3，就此，安装记录已经删除，然后请重复方法一的步骤。

心得：对于这类不需要连接服务器进行验证的限期使用软件，都可以采用类似的方法破解。
1，软件必须要在系统中找一个地方存储安装信息。
2，在mac文件系统里找到这个地方。
3，删掉生成的时间记录。
4，重新运行。

  2. 修改 trialStart3 为当前时间

popclip

popclip.plist 在 preference 中

在popclip.plist中的TrialData项，原本为Data格式，切换成Binary Plist之后可以打开一个子设置项，找到剩余次数的选项，修改为任意数值，生成为一个新的plist文件，在用文本编辑器打开，复制并替换原本数据完成破解。
